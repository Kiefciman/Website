## Mymangastats Cli tool that helps you organize your manga. You can edit the metadata of mangas such as its state (reading/finished/dropped/on hold/plan to read), chapters, progress. Mymangastats uses a txt file (~/.config/manga.txt) as a manga database, but you don't need to edit this file, because the script takes care off adding new manag or editing existing ones. ![](https://raw.githubusercontent.com/Kiefciman/mymangastats/main/image.png) ### How to install - **Linux:**
Dependencies: fzf git
Debian/Ubuntu based distros: ``` sudo apt install fzf git ``` Suse/Fedora based distros: ``` sudo dnf install fzf git ``` Arch based distros: ``` sudo pacman -S fzf git ``` Then clone this repo: ``` git clone https://github.com/Kiefciman/mymanga.git ``` - **Windows:**
Dependencies: bash fzf git
We are going with scoop
First install scoop: ``` Set-ExecutionPolicy RemoteSigned -Scope CurrentUser irm get.scoop.sh | iex ``` Then we install fzf and git (this will also install bash): ``` scoop install git scoop update scoop install fzf ``` Then open the Bash shell by typing **bash** or set bash as a terminal profile (at the path simply type bash)
Then clone this repo: ``` git clone https://github.com/Kiefciman/mymanga.git ``` - **Android:**
Not tested yet ### To run the script `./mymangastats.sh` or `/path/to/mymangastats.sh`
